# Megnacodeit3.0
## Moon Phase Calendar

This project is a **Python-based Moon Phase Calendar** that visually displays daily moon phases for a selected month and year. It uses the **ephem** library for accurate lunar calculations and **Python Turtle graphics** to render each moon phase along with its date in a calendar-style layout.
